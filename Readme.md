# HODU Landing Page (Responsive)

고양이 이미지를 활용한 **랜딩 페이지(홈페이지 메인)**를 HTML/CSS로 구현한 프로젝트입니다.  
상단 고정 헤더(Header)부터 Hero 섹션, Gallery 섹션, 구독(Subscribe) UI, 푸터(Footer)까지 하나의 흐름으로 구성되어 있으며, **데스크톱/모바일 반응형 레이아웃**을 적용했습니다.

---

## 🔗 프로젝트 개요

- **프로젝트 목적**
  - Figma 시안을 바탕으로 html 마크업하고, CSS로 레이아웃/스타일을 구현한다.
  - Flex / Grid 레이아웃을 활용하여 각 섹션에 맞는 UI를 구성한다.
  - 미디어쿼리를 활용하여 모바일 환경(768px 이하)에서 레이아웃이 자연스럽게 변하도록 만든다.

- **핵심 구현 포인트**
  1. `header` 상단 고정 
  2. Hero 섹션: 텍스트/이미지 2컬럼 구조 → 모바일에서 세로 배치 전환
  3. Gallery 섹션:  
     - Gallery-1: 이미지/텍스트 좌우 배치 (Flex)  
     - Gallery-2: 이미지 3개를 Grid로 배치 → 모바일에서 1열로 전환  
     - Gallery-3: 구독 박스를 카드 형태로 배치(배경색/둥근 모서리/인풋 UI)
  4. 모달창 구현 (이건 완성하지 못했습니다)

## 🛠 사용 기술

- **HTML5**
  - 시맨틱 태그(`header`, `nav`, `main`, `section`, `footer`)를 중심으로 문서를 구조화
- **CSS3**
  - 유지보수성을 위한 CSS 변수(`:root`)활용 : 컬러/컨테이너 폭 관리
  - Flexbox: 헤더/히어로/갤러리 레이아웃 구성
  - Grid: 이미지 갤러리 3열 구성
  - Media Query: `max-width: 768px` 기준 반응형 처리
- **Reset CSS**
  - 브라우저 기본 스타일 차이를 줄이기 위해 reset.css 사용

## 🧩 페이지 구성

### 1) Header (고정 헤더)
- 왼쪽: 로고 이미지
- 오른쪽: 내비게이션 링크(Home, Support) + Download 버튼
- 스크롤을 내려도 헤더가 화면 상단에 고정되도록 처리했습니다.(position:fixed와 z-index)

### 2) Hero Section
- 랜딩 페이지 첫 화면 영역
- 텍스트 박스(제목/설명/Download 버튼) + 이미지 박스(고양이 이미지)를 좌우로 배치
- 데스크톱: 좌우정렬
- 모바일: 세로 배치로 전환 + 텍스트 중앙 정렬

### 3) Gallery Section
#### Gallery-1
- 큰 이미지 1개 + 설명 텍스트 구조
- Flex로 좌우 정렬, 모바일에서는 세로 정렬로 변경

#### Gallery-2
- 상단: 제목(h3) + 설명 문단
- 중단: 3개 이미지 갤러리(Grid 3열)
- 하단: 설명 문단 + Learn More 버튼(가로 배치)
- 모바일에서는 Grid를 1열로 바꾸고, 버튼 영역도 세로로 변경했습니다.

#### Gallery-3 (Subscribe)
- 배경색이 다른 강조 카드 UI
- 왼쪽: 구독 안내 텍스트
- 오른쪽: 이메일 입력창 + Subscribe 버튼
- 둥근 모서리, 인풋 영역 화이트 배경으로 UI 분리

### 4) Footer
- 왼쪽: 로고
- 오른쪽: SNS 아이콘 링크 4개(블로그/인스타/페이스북/유튜브)

## ✅ 반응형 동작 정리

- **768px 초과(데스크톱)**
  - 헤더 네비게이션 표시
  - Hero: 텍스트/이미지 2컬럼
  - Gallery-2: 이미지 Grid 3열

- **768px 이하(모바일)**
  - 네비게이션 숨김(`.navbar { display:none; }`)
  - Hero: 세로 레이아웃 전환 + 텍스트 중앙 정렬
  - Gallery-1: 세로 전환
  - Gallery-2: 이미지 Grid 1열 전환 + 버튼 영역 세로 전환
  - 전체 폰트/간격을 모바일에 맞게 축소
